# Adult Income Data Analysis (Python EDA)
This repository contains the code and documentation for an Exploratory Data Analysis (EDA) of Adult Income data using Python.

## Dataset

The dataset used for this analysis can be found [here](https://www.kaggle.com/datasets/wenruliu/adult-income-dataset). It includes information about adult income. Here are the columns in the dataset:

- **age**: Age of the individual.
- **workclass**: Type of workclass (e.g., Private, Self-emp-not-inc).
- **fnlwgt**: -
- **education**: Educational level.
- **educational-num**: Educational level as a numerical value.
- **marital-status**: Marital status.
- **occupation**: Occupation of the individual.
- **relationship**: Relationship status.
- **race**: Race of the individual.
- **gender**: Gender of the individual.
- **capital-gain**: Capital gain.
- **capital-loss**: Capital loss.
- **hours-per-week**: Number of hours worked per week.
- **native-country**: Native country of the individual.
- **income**: Income category (e.g., <=50K, >50K).

## EDA File

[Adult_Income.ipynb](Adult_Income.ipynb): Jupyter Notebook containing the Python code for the exploratory data analysis of Adult Income data.

## Usage

You can clone this repository to your local machine and run the Jupyter Notebook to perform the EDA. Make sure you have the required Python libraries installed.

```bash
git clone https://github.com/HloHarshit/Adult-Income-Python-EDA.git
cd Adult-Income-Python-EDA
jupyter notebook Adult_Income.ipynb
```
